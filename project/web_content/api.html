API Usage

http://www.shopasource.com/api/shop/search?sk={keyword}&smatch={match_accuracy}&shl=true&slh=false&shops={shop_name}

  • Match Accuracy - refine results based on keyword
  • SHL - High to low
  • SLH - Low to High
  • Shop - Shop Name, single shops http://www.shopasource.com/api/shop/search?sk=wallet&smatch=8&shl=true&slh=false&shops=AMAZON
  • Shop - Shop Name, multiple shops [AMAZON, TARGET] http://www.shopasource.com/api/shop/search?sk=wallet&smatch=8&shl=true&slh=false&shops=[AMAZON, TARGET]
  • Keyword - search keyword

Go to http://www.shopasource.com/websearch/shops-active.json to see list of shops available
